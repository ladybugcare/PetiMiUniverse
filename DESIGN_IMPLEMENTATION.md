## 🎨 Design System Features

### **Colors**
- **Primary Purple**: #a855f7 (bright purple)
- **Secondary Purple**: #9333ea (deep purple)
- **Background Light**: #f5f3ff (very light purple)
- **Text Dark**: #262626 (neutral dark)
- **Text Medium**: #525252 (neutral medium)
- **White**: #ffffff

### **Typography**
- **Display Font**: Poppins (bold/semibold, for headings) - Moderna, geométrica e profissional
- **Body Font**: Inter (regular, for text) - Excelente legibilidade e clareza
- **Sizes**: Responsive from 1rem to 7rem
- **Aplicação**: `.text-display` usa Poppins, body e parágrafos usam Inter

### **Spacing**
- Consistent spacing scale (0.25rem to 6rem)
- Large padding for sections (5rem = 80px)
- Generous gaps between elements

### **Components**
- **Icon Cards**: White background, purple gradient icons, hover lift effect
- **Timeline Steps**: Circular badges with numbered indicators
- **Buttons**: White buttons on purple background with hover effects
- **Footer**: Multi-column responsive layout

### **Animations**
- Fade-in-up animation for hero content
- Hover lift for cards (-8px translateY)
- Scale transform for icons (110% on hover)
- Smooth transitions (150ms to 350ms)

## 📱 Responsive Breakpoints

- **Mobile**: < 640px (single column)
- **Tablet**: 640px - 1024px (2 columns)
- **Desktop**: > 1024px (3-4 columns)
