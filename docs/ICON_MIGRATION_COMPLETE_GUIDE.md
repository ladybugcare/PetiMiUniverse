## 🎨 Padrão de Implementação

### 1. Para Menu Items (Sidebar)
```tsx
{
  id: 'item-id',
  label: 'Label',
  icon: <IconName size={20} color={colors.primary} />,
  action: 'navigate',
  path: '/path'
}
```

### 2. Para Títulos
```tsx
<h1>
  <div style={{ display: 'flex', alignItems: 'center', gap: '12px' }}>
    <IconName size={32} color={colors.primary} />
    <span>Título</span>
  </div>
</h1>
```

### 3. Para Dicas/Info Inline
```tsx
<p style={{ display: 'flex', alignItems: 'center', gap: '6px' }}>
  <Info size={16} color={colors.primary} />
  Texto da dica
</p>
```

### 4. Para Validação
```tsx
{isValid && (
  <span className="absolute right-3 top-1/2 -translate-y-1/2 text-green-500">
    <CheckCircle size={20} />
  </span>
)}
```

### 5. Para Botões
```tsx
<button>
  <IconName size={18} />
  <span>Label</span>
</button>
```

### Para substituir todos os emojis de uma vez:

```bash

find . -name "*.tsx" -o -name "*.ts" | while read file; do
  # Substituir emojis comuns
```
